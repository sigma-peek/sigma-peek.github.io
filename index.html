Tutoriál, jak hostovat svou webovou stránku přes GitHub

1. Vytvoření repositáře na GitHub

  1. Zvolíme jméno repositáře.
  2. Můžeme nebo nemusíme přidat popis.
  3. Nastavíme přístupnost.
  4. Přidáme uvedení (pouze v případě že neimportujeme existující repositář).
  5. Vytvoříme repositář.

Poznámka: Pro vytvoření repositáře musíte mít vytvořený účet a být přihlášeni na GitHub.com.

2. Získáme kopii našeho depositáře

  1. Otevřeme si náš nově vytvořený depositář.
  2. Klikneme na „code“.
  3. Zkopírujeme si odkaz, který se nám zobrazí.
  4. Otevřeme si příkazový řádek a přesuneme se do složky, kam chceme stáhnout kopii repositáře.
  5. Zadáme příkaz: git clone [Váš odkaz].
  6. Pomocí příkazového řádku se dostaneme do našeho repositáře pomocí příkazu: cd [název Vašeho repositáře].
  7. Nakonec otevřeme repositář pomocí příkazu: code ..

Poznámka: Pro otevření vašeho repositáře v počítači, musíte mít nainstalovaný Visual Studio Code a pro stažení repositáře musíte mít nainstalovaný GitHub.

3. Tvorba html stránky

V repositáři si vytvoříme soubor index.html.

Napíšeme „doc“ a dáme enter.

Uvnitř <body> si vytvoříme vzhled naší html stránky.

Poznámka: Po každé provedené úpravě je potřeba všechnu práci uložit.

4. Přesunutí naší stránky na web

V horní části GitHubu klikneme na Terminál.

V dolní části se nám otevře terminál a zadáme několik příkazů.

  1. Příkaz, který nám bude přidávat změny: git add .
  2. Příkaz, který bude předkládat změny: git commit -m initialcommit
  3. Příkaz, který protlačí změny: git push origin main

Nyní může přejít zpět na GitHub a zkontrolovat, jestli všechny změny proběhly úspěšně.

Pro hostování webové stránky přejděte do nastavení ve vašem projektu na GitHub.com.

V oddělení GitHub Pages nastavte zdroj na „main“ a uložte.

Znovu najeďte na GitHub Pages a nyní by zde měl být viditelný odkaz na vaší webovou stránku.

5. Designování naší webové stránky

Vrátíme se do Visual Studio Code.

Do Terminálu zadáme příkaz: git checkout -b [libovolný název]

Vytvoříme nový soubor styles.css.

Odkážeme na tento soubor v index.html, tím že, do <head> pod <title> napíšeme link a dáme enter.

Do nově vygenerovaného textu vložíme pouze styles.css takto: href=“styles.css“

Nyní si můžeme vytvořit design naší stránky ve styles.css.

Po dokončení designu uložíme a musíme znovu protlačit naší práci na naší stránku.

Do terminálu opět zdáme stejné příkazy jako před tím, ale místo initialcommit zadáme název který jsme si zvolili na začátku tvorby designu: git add . ; git commit -m [Váš zvolený název] ; git push origin [Váš zvolený název]

Vrátíme se zpět na GitHub.com a v našem repositáři v horní části můžeme vidět oznámen s tlačítkem „Compare & pull request“, klikneme na toto tlačítko.

V dolní části se zobrazí tlačítko „Create pull request“, klikneme na něj.

Chvíli počkáme a objeví se další tlačítko „Merge pull request“ a znovu na něj klikneme a potvrdíme.

Přejdeme zpět na naší stránku a obnovíme ji.

Pokud jste vše udělali správně, měl by se aplikovat design, který jste si vytvořili.

A toto je vše, co potřebujete vědět, abyste mohli hostovat svou stránku přes GitHub.
